# Tâche 14.5 - Accessibilité WCAG AA

**Epic :** 14 - UX/UI Polish & Accessibilité
**Priorité :** Haute
**Statut :** [ ] À faire

## Actions à réaliser

1. **Labels ARIA :** Ajouter aria-label sur les boutons icône, aria-describedby sur les formulaires
2. **Navigation clavier :** Tab order logique, focus visible, Escape pour fermer les modals
3. **Contraste couleurs :** Vérifier que tous les textes ont un ratio > 4.5:1
4. **Screen reader :** Tester avec NVDA/VoiceOver, ajouter les rôles ARIA manquants
5. **Images :** Alt text sur toutes les images

## Critères d'acceptation

- [ ] Score Accessibility Lighthouse > 90
- [ ] Navigation clavier complète
- [ ] Contraste suffisant partout
- [ ] Compatible screen reader
