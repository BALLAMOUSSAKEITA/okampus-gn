# Tâche 14.1 - Design system composants

**Epic :** 14 - UX/UI Polish & Accessibilité
**Priorité :** Haute
**Statut :** [ ] À faire

## Actions à réaliser

1. **Créer des composants réutilisables :**
   - `Button` : variants (primary, secondary, outline, danger), sizes (sm, md, lg), loading state
   - `Card` : avec header, body, footer optionnels
   - `Modal` : avec overlay, close button, responsive
   - `Input` : avec label, error message, helper text
   - `Badge` : variants colorées pour les statuts
   - `Toast` : (déjà créé dans Epic 02)
   - `Avatar` : image ou initiales, tailles variées
   - `Tabs` : navigation entre onglets

2. **Documenter les composants** dans un fichier de référence

## Critères d'acceptation

- [ ] Chaque composant est réutilisable avec des props typées
- [ ] Les variants sont cohérentes avec le design existant
- [ ] Les composants sont utilisés sur toutes les pages
