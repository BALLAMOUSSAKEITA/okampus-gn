## Critères de validation

- Les fichiers sont uploadés et stockés de manière persistante
- Les types MIME sont validés
- La taille maximale est respectée
- Les ressources premium nécessitent un achat pour le téléchargement
- Les avatars sont affichés dans le profil et la navbar
