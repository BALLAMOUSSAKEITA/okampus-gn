# O'Kampus - Propositions d'améliorations

## Vision : Plateforme complète de l’étudiant guinéen

Du **succès au bac** → **orientation** → **pendant les études** → **insertion professionnelle**

---

## ✅ Fonctionnalités déjà implémentées

1. **Assistant IA d'orientation** – Analyse projet, forces, faiblesses, notes → recommandations personnalisées
2. **Inscription** – Bachelier ou Étudiant
3. **Étudiants conseillers** – Possibilité de devenir conseiller avec description personnalisée
4. **Chat avec conseillers** – Discussion en direct par filière
5. **Rendez-vous Meet** – Prise de RDV pour appels vidéo avec les conseillers
6. **Forum** – Questions sur universités, filières, débouchés

---

## 🚀 Améliorations proposées

### Phase 1 : Préparation au Bac

| Fonctionnalité | Description |
|----------------|-------------|
| **Ressources Bac** | Fiches de révision, annales, sujets des années précédentes par série |
| **Calendrier des examens** | Dates du bac, concours d'entrée, inscriptions |
| **Quiz de révision** | Tests par matière pour s'auto-évaluer |
| **Conseils méthodologie** | Articles/vidéos : comment réviser, gérer le stress, s'organiser |
| **Simulateur de notes** | Estimation des notes possibles selon les coefficients |

### Phase 2 : Pendant les études

| Fonctionnalité | Description |
|----------------|-------------|
| **Suivi de parcours** | Tableau de bord : notes, progression, objectifs par semestre |
| **Calendrier académique** | Rentrée, examens, vacances par université |
| **Ressources par filière** | Syllabi, cours, supports partagés par les étudiants |
| **Réseau étudiant** | Connexion entre étudiants de même filière/université |
| **Groupes d'étude** | Créer ou rejoindre des groupes pour réviser ensemble |
| **Alertes et rappels** | Notifications : échéances, examens, inscriptions |

### Phase 3 : Insertion professionnelle

| Fonctionnalité | Description |
|----------------|-------------|
| **Offres de stage** | Annonces de stages et jobs étudiants en Guinée |
| **Bourses et opportunités** | Base des bourses, concours, programmes d'échange |
| **Témoignages diplômés** | Parcours de diplômés guinéens (secteur, parcours, conseils) |
| **CV et portfolio** | Création de CV, portfolio pour les filières créatives/tech |
| **Préparation entretiens** | Conseils et simulations pour les entretiens de stage/emploi |

### Phase 4 : Communauté et contenu

| Fonctionnalité | Description |
|----------------|-------------|
| **Blog / Actualités** | Articles sur l'éducation en Guinée, réformes, nouveautés |
| **Statistiques** | Taux de réussite par filière, université, débouchés |
| **Carte des universités** | Carte interactive des établissements en Guinée |
| **Comparateur de filières** | Comparer durée, coût, débouchés, difficulté |
| **Système de badges** | Récompenses : "Conseiller actif", "Contributeur forum", etc. |

### Phase 5 : Technique et expérience

| Fonctionnalité | Description |
|----------------|-------------|
| **Application mobile** | PWA ou app native (React Native) pour accès mobile |
| **Notifications push** | Rappels RDV, réponses forum, messages conseillers |
| **Intégration Meet/Zoom** | Génération automatique de liens Meet pour les RDV |
| **IA avancée** | Connexion OpenAI/Claude pour conseils plus personnalisés |
| **Authentification sécurisée** | OAuth, 2FA, récupération de mot de passe |
| **Backend complet** | Base de données (Supabase, Firebase), API REST |
| **Modération** | Modération du forum, signalement, règles de communauté |

### Phase 6 : Partenariats

| Fonctionnalité | Description |
|----------------|-------------|
| **Partenariat universités** | Validation des infos, accès aux actualités officielles |
| **Partenariat ONG/Ministère** | Visibilité, soutien, bourses |
| **Sponsoring** | Entreprises guinéennes pour financer la plateforme |
| **Ambassadeurs** | Étudiants relais dans les lycées pour promouvoir O'Kampus |

---

## 📊 Priorisation suggérée

1. **Court terme** : Backend + Auth réelle, Intégration Meet, Ressources Bac
2. **Moyen terme** : Suivi de parcours, Offres de stage, Application mobile
3. **Long terme** : IA avancée, Partenariats, Statistiques publiques

---

## 🎯 Différenciation O'Kampus

- **100% focalisé sur la Guinée** (universités, concours, contexte local)
- **Pair-à-pair** : étudiants qui conseillent les bacheliers
- **Parcours complet** : du lycée à l'insertion professionnelle
- **Gratuit et accessible** : pas de barrière financière
- **Communauté francophone** : langue et culture partagées
